Pricing Information
How It Works
  • Real-time Pricing: Fetches current prices from supplier websites
  • Intelligent Caching: Stores pricing for 24 hours to avoid repeated requests
  • Fallback System: Uses cached pricing if website is unavailable
  • Multiple Sources: Tries web scraping, then database, then estimates
Supported Suppliers
  • Ubiquiti UniFi: Real pricing from ui.com
  • Cisco: Estimated pricing (requires login for real pricing)
  • Connectix: Estimated pricing for common products
  • Other Suppliers: Database pricing or estimates
Tip: The system now uses real pricing from supplier websites instead of AI estimates. For example, U6-Pro Access Points now show £125.00 (from Ubiquiti website) instead of estimated £220.00.
